我现在任职于平安科技投研与咨询团队。我在 2019 年从复旦大学博士毕业后来到平安科技的，部门服务对象主要为平安集团上层、政府单位（比如深圳市政府、统计局）以及外部的一些合作单位（比如金茂集团等）。我在 2019 年底转到房地产组之后主要做的是商业地产、商业住宅相关的研究。因为房地产是一个科技渗透率非常低的行业，早期我们对于行业的研究都是遵循业界的范式，围绕变量之间相关性、周期定位、周期拐点展开，预测使用的多为 ARIMA、VAR 等传统时序计量方法。后来我们逐渐把一些机器学习的方法应用到我们的项目中，主要项目有基于随机森林模型的房地产政策预测以及长租公寓租金定价模型、基于类重复交易模型的写字楼资产价格指数等。在此期间，我实现了从纯计量方法到机器学习的转变，也让我对未来的方向（数据科学家）更加明确。

项目经历

城市写字楼租金预测——基于领先性质的主成分因子模型

2019.10-2020.1

从宏观、中观因子中筛选能够对于 4 个主要城市写字楼租金具有预测性的因子并做分类，检验每一类别下因子对于城市写字楼租金的领先性质与领先期数，根据因子的领先性进行特征工程，使用 PCA 对于每一类因子构建主成分因子 区分正负相关性 ，根据主成分因子和相关性，利用 sk-learn 对租金进行回归建模。成果：交付给平安不动产租金预测的文档、代码和城市 板块未来 1 年和 5 年的预测结果。

房地产政策预测——基于非平衡样本的随机森林模型

2020.11-2021.4

我们构建了房地产政策 4 级标签体系，将从 2015 年以来的各地房地产政策进行标签化，将非结构化数据转为结构化数据，在此基础上构建了房地产调控指数；其次与平安欧拉图谱团队 ，对历史房地产政策进行情感标注，结合标签体系，对各个城市房地产的数据（如新房、二手房、房价租金比、去化率等）做特征工程，使用 Logit Regression、Random Forest 和 XGBoost 等模型对各个城市房地产政策进行分类预测，在考虑正负样本的非平衡性后，最终 Balanced Accuracy 为 86% 左右。

写字楼投资基准——基于 PSM 的价格指数

2021.5-2021.12

写字楼的投资周期非常长，成本法难以计量写字楼投资的收益，因此平安需要写字楼投资基准作为投资参考。相同物业在多次交易时能够反映跨时的价格增长/指数；但写字楼交易数据比较少，因此我们尝试构建了 Pseudo Repeat Sales Model，在一个更大的匹配空间假定物业的同质性与可比性，并结合 Hedonic Model 去除写字楼自身特征的影响，构建写字楼投资基准/价格指数。
